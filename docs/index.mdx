## Why Copy-Paste?

Most UI libraries lock you into their design system. With Arctis UI, the code **lives in your project** the moment you paste it. That means:

- **Full ownership** â€” tweak anything, anytime, without fighting library internals
- **Zero runtime dependency** â€” no extra bundle weight from a library you don't fully use
- **Framework agnostic-friendly** â€” works anywhere React runs
- **Readable source** â€” every component is clean, commented, and meant to be understood
